README — What This Is
## What This Is

A 3-layer Claude Skill system that helps anyone — no matter their
industry or role — package real work experience into shareable,
reusable knowledge products.

Most people underestimate the value of what they know. This skill
system changes that by guiding you to:

- Surface the real lessons buried in your day-to-day work
- Structure them into something others can learn from
- Output them as a document, spreadsheet, PDF, or interactive site

README — The Core Idea
## The Core Idea

You don't need to be an expert to share something valuable.
You just need to be 2–3 steps ahead of someone who's where
you were a year or two ago.

Your mistakes, your workarounds, your hard-won judgment calls —
none of that exists in an AI's training data.
That's exactly what makes it worth packaging.

README — Three-Layer Architecture
## Three-Layer Architecture

Layer 1 · experience-to-asset (main entry)
  → Lowers the barrier to entry
  → Shows users the 'frame' before asking any questions
  → Decides when to move deeper

Layer 2 · experience-deep-extract
  → Draws out real stories, mistakes, and judgment calls
  → One question at a time, conversational not interrogative
  → Combines spoken experience with uploaded documents

Layer 3 · experience-package-build
  → Matches content to the right output format
  → Follows: Audience → Value Promise → Density → Format → Build
  → Generates the actual deliverable (Excel, PDF, HTML, article...)

README — How to Install
## How to Install

1. Download `experience-to-asset.skill` from the Releases page
2. Go to Claude.ai → Settings → Skills
3. Upload the .skill file
4. Start a new conversation and say something like:
   'I've been doing BD for a year, help me package my experience'

## Real Example

This skill system was built using its own methodology.

Input: A year of TikTok Shop creator operations in Thailand —
raw Excel tracking sheets, activation campaign records,
live-streaming runbooks, and team onboarding notes.

Output:
- BD Operations Handbook (Excel, 6 sheets, real data)
- Live Streaming Operations Manual (Excel, 6 sheets)
- BD Growth Map (interactive HTML website)
- This skill system itself
